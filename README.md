## 📌 About The Project

Urban Pulse is a modern startup landing page developed as **Project 2 of the Navyan Web Development Internship**.

The project showcases a smart-city platform that empowers citizens and local authorities through technology, real-time analytics, issue reporting, and sustainable urban development.

The goal was to build a professional, responsive landing page similar to real-world startup websites while following modern web design principles.
